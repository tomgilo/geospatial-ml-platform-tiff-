# 🌍 地理空间TIFF机器学习平台

> **栅格数据一键建模 | 12种算法任你选 | 年月双模 | 自带SHAP解释 | 模型还能复用**

把一堆地理空间 TIFF 栅格数据扔进去，选几个模型，点一下按钮，然后坐等预测图、评估报告、SHAP 可解释性分析全自动端上来。

---

## 🎯 这玩意儿是干啥的？

一个基于 Streamlit 开发的桌面端 Web 应用，专门解决一个问题——**让你手里的逐年/逐月 TIFF 栅格数据，从"躺在硬盘里吃灰"到"变成论文级分析图表"，只需要点几下鼠标。**

适用人群：遥感地信、气象研究、生态建模、以及任何手握一堆 TIFF 栅格数据不知道咋处理的科研同行。

## ✨ 核心功能

### 🔬 12 种机器学习模型

| 门派 | 模型 | 特点 |
|------|------|------|
| **线性派** | OLS、Ridge、Lasso、ElasticNet | 简单粗暴、跑得快、可解释性强 |
| **树模型派** | RF、XGBoost、LightGBM、ExtraTrees、GBR | 非线性建模猛将，特征重要性自动排名 |
| **其他派** | SVR、MLP、KNN | 支持向量、神经网络、K近邻，玩花的来这几个 |

### 📅 年度 & 月度双模式

- **年度数据**：`2000.tif`、`2001.tif` 直接读
- **月度数据**：平铺式 `2000-01.tif` 或嵌套式 `2014/01.tif`，系统自动识别

### 🔮 模拟 & 外推双预测模式

- **模拟模式**：有未来 X 数据？直接用实际值预测
- **外推模式**：没有未来 X？拿训练末年数据兜底

### 🧠 SHAP 可解释性分析

老板/审稿人问"凭什么这个变量重要"？SHAP 值直接甩脸上——特征重要性排序图、蜂群图、依赖图，量化得明明白白。

### 🔄 模型一键复用

训练完自动保存（joblib 格式），下次直接加载 + 喂新数据，**无需重新训练**。

### 📊 全自动出图

散点图、残差图、RMSE 箱线图、特征重要性柱状图、空间 R²/RMSE 分布图……全部自动生成 PNG，**直接拖进论文**。

### 📝 报告一键导出

- HTML 报告浏览器直接预览
- PDF 报告一键下载
- CSV 表格全量导出（逐像素指标、特征重要性）
- JSON/YAML 配置记录，方便复现

### 🔍 逐像元时间序列

随便点一个像素，完整时间序列 + 所有模型预测值叠加对比，灰色竖线标注训练/预测分界。

## 🚀 快速上手

### 环境要求

- Windows 10/11
- 内存 8GB+（CPU 核心越多训练越快）
- **软件自带嵌入式 Python 3.12，零依赖安装，解压即用**

### 三步出图

```
第1步：填入 Y（因变量）和 X（自变量）的 TIFF 文件夹路径
第2步：选择数据模式 + 时间范围 + 勾选模型
第3步：点击「开始训练」，坐等结果
```

### 启动方式

- 双击 `启动应用.vbs`——无黑窗、无命令行、双击即开
- 或 `启动应用.bat` 命令行方式启动
- 首次启动自动打开浏览器访问 `http://localhost:8501`

## 📁 数据格式要求

- **格式**：单波段 GeoTIFF（`.tif` / `.tiff`）
- **对齐**：所有 TIFF 必须相同空间分辨率、投影坐标系、行列数
- **Nodata**：自动识别并转 NaN

### 命名规则

| 数据类型 | 命名格式 | 示例 |
|---------|---------|------|
| 年度 | `YYYY.tif` | `2014.tif` |
| 月度（平铺） | `YYYY-MM.tif` | `2014-01.tif` |
| 月度（嵌套） | `YYYY/MM.tif` | `2014/01.tif` |

## 📦 输出文件

```
outputs/
├── predictions/      # 预测结果 TIFF（直接拖进 ArcGIS/QGIS）
├── charts/           # 可视化图表 PNG（论文插图不用愁）
├── tables/           # 数据表格 CSV（二次统计随便折腾）
├── reports/          # HTML/PDF 评估报告
├── models/           # joblib 模型文件 + metadata.json
└── config.json       # 运行配置记录（方便复现）
```

## 🧪 自带测试数据

`test_data/` 目录包含 6 个变量（LAI、SSR、SRC、T2m、TP、DEM），覆盖 2014–2023 年，拿到就能直接跑。
